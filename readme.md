# 工程简介
因为一些原因，需要学习`Qt + openGL`，而使用`Qt`时，不建议直接引用`OpenGL`的源码头文件，这会影响到跨平台特性啥的。因此本工程的目的是通过实现一个简单的`*.obj`模型浏览器来学习Qt自己的OpenGL渲染。

## 环境
本人试图不引用其他库，只用`Qt`来做，如果后面不得不引用其他库时，我会在下面列出。
```
Qt版本:   Qt 5.8 msvc2015
操作系统:  windows 10
```
